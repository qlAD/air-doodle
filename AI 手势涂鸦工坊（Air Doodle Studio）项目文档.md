# AI 手势涂鸦工坊（Air Doodle Studio）项目文档

## 1 项目概述

### 1.1 项目简介

《AI 手势涂鸦工坊》是一款**基于浏览器摄像头、前端 AI 识别、Next.js 全栈架构**的软硬件结合交互式创意项目。项目仅依托电脑、手机内置摄像头作为唯一硬件采集设备，通过 WebRTC 技术实时获取设备视频流，结合 MediaPipe WASM 轻量化手部 AI 模型，在浏览器本地完成手势识别运算，实现零设备、无接触的隔空涂鸦绘画体验。

本项目彻底摒弃传统鼠标、数位板、触屏的交互方式，打造纯体感手势绘画模式。依托 Next.js 全栈开发能力，实现画作云端存储、在线画廊展示、作品社区互动、模板临摹创作、体感趣味小游戏、一键社交分享等全流程完整功能。项目全程基于浏览器运行，跨终端、跨系统兼容，无需下载安装客户端，兼具趣味性、工程实践性与技术创新性，可广泛应用于高校课程设计、全栈开发实训、科创竞赛作品展示等场景。

项目对外统一品牌标识为 Air Doodle，代码仓库命名为 air-doodle，英文全称为 Air Doodle Studio；官方中文名称固定为《AI 手势涂鸦工坊》，所有作品导出、分享海报、视频水印均统一使用品牌名 Air Doodle，形成标准化品牌视觉体系。

### 1.2 项目开发背景

当前市面上传统手势涂鸦方案大多基于 Python 本地桌面程序开发，存在诸多技术短板与体验缺陷：项目高度依赖本地运行环境，仅支持单机离线使用，无法实现作品云端存储与在线分享，功能拓展性极差。同时，传统电子白板、手绘工具交互模式单一，人机交互生硬刻板，缺乏沉浸式体验与趣味性。

随着 WebAI、WebRTC、WASM 前端技术的快速迭代与成熟，现代浏览器已具备离线 AI 计算、实时视频流处理、轻量化模型部署的完整能力。基于 Next.js 全栈框架重构手势涂鸦项目，可实现**前端本地 AI 识别 + 后端数据持久化**的标准化 Web 应用架构，彻底解决传统本地项目的环境依赖、功能单一、无法线上部署等核心痛点，打造可迭代、可部署、可商业化的现代化 Web 创意项目。

除此之外，现有同类手势涂鸦项目普遍存在**玩法单一、社交属性缺失、新手入门门槛高、用户激励体系空白**的问题，用户体验碎片化，普遍存在短期游玩后快速流失的现象。本项目针对行业痛点进行系统性、全方位升级，从单一的“手势绘画工具”迭代升级为**轻量化体感互动娱乐 Web 应用**，整合自由涂鸦、模板临摹、体感小游戏、双人同屏协作、作品社区互动等多元玩法，大幅提升用户使用时长与自发传播能力，完善产品闭环。

### 1.3 项目创新点

1. **轻量化软硬件结合**：仅以设备自带摄像头为唯一硬件采集设备，依托通用 Web 标准完成图像采集与交互，零额外硬件成本、全设备跨端适配，形成完整轻量化软硬件交互闭环。

2. **前端离线 AI 计算**：基于 MediaPipe WASM 轻量化模型，在浏览器本地完成全部手势识别运算，原始视频流无需上传服务器，兼顾 100ms 以内低延迟交互、用户隐私安全与服务器算力减负三大优势。

3. **Next.js 全栈一体化架构**：依托单一 Next.js 框架，统一实现客户端交互渲染、服务端 SSR 渲染、API 接口开发、数据库数据存储、项目静态部署，工程结构规范、代码解耦度高，便于迭代与维护。

4. **全手势无键鼠自研交互体系**：自主设计分层手势优先级规范，实现全域无鼠标、无触屏纯体感操控，页面导航、绘画创作、画布漫游、素材调用、截图导出、参数调节全部依托手势完成，交互自由度与科技感远超传统同类涂鸦项目。

5. **数据持久化与在线展示**：突破传统单机涂鸦无留存、无分享的短板，支持作品云端保存、在线画廊分页展示、个人作品管理，实现创作内容永久留存与线上公开展示。

6. **零门槛智能临摹创作体系**：内置海量分层临摹模板库，涵盖卡通人物、小动物、星空风景、简约简笔画、节日主题素材等多类型资源，以半透明底图叠加于画布底层，新手可沿轮廓快速临摹出完整画作，大幅降低创作门槛，临摹完成自动解锁成就勋章。

7. **多模式体感娱乐闭环**：打破单一绘画工具定位，融合自由涂鸦、分级模板临摹、多款体感小游戏，覆盖零基础娱乐用户与专业创意创作者，有效拉长用户停留时长，完善产品娱乐属性。

8. **双人多手协同交互**：基于 MediaPipe 多手同步检测能力，支持同一设备同时识别双人、多只手掌，实现亲子、情侣同屏共创绘画、趣味对战，线下互动性与场景适配性极强。

9. **视听一体化体感反馈**：配备星光、彩虹、烟花、水彩晕染、荧光霓虹等动态特效笔刷，结合全屏烟花动画、场景专属音效、移动端触觉震动反馈，全方位强化无接触交互的沉浸感与氛围感。

10. **完整社交传播链路**：搭建作品社区生态，支持作品投稿、点赞、收藏、评论、复刻临摹、热门榜单展示；支持一键生成品牌分享海报与 3-10 秒绘画过程短视频，自带专属品牌水印与文案模板，支持多平台下载与分享，具备天然自发传播属性。

11. **层级化成长激励系统**：设计连续打卡、创作数量、小游戏通关、笔刷解锁等多元成就体系，解锁专属限定素材与特效，搭配节日限时主题活动，提升用户复访频率与长期粘性。

12. **无限延伸画布交互**：采用底层画面固定、上层涂鸦图层独立漫游的架构，支持手势拖拽无限拓展画布，突破传统固定画布尺寸限制，实现无边界自由创作。

### 1.4 运行环境

#### 硬件环境

搭载可用摄像头的台式电脑、笔记本电脑、智能手机，设备需支持 WebRTC 多媒体通信标准，可正常采集实时视频流。

#### 软件环境

操作系统：Windows、macOS、Android、iOS 全平台适配；主流浏览器：Chrome、Edge、Safari 最新稳定版本；开发环境：Node.js 18 及以上、Next.js 15。

## 2 系统需求分析

### 2.1 功能需求

#### 2.1.1 高阶全手势核心交互功能

项目摒弃传统键鼠、触屏操作，实现**100% 全域手势操控**，覆盖页面导航、画布操控、绘画创作、素材调用、截图导出、参数调节全流程场景。通过自研手势分层互斥、防误触逻辑，适配无限延伸画布架构，实现手势状态自动切换、无冲突响应。

**基础模式自动切换（核心底层逻辑）**

- 光标模式（默认闲置）：单独食指伸直、其余手指弯曲收拢，等效鼠标光标，支持隔空移动、单击选中、悬停唤起右键菜单、双指比耶滑动滚动页面，仅负责操控，不生成绘画笔迹。

- 绘画模式：食指、中指并拢伸直、其余手指弯曲收拢，以中指指尖为笔尖持续生成平滑线条，中指收回即刻终止绘画，自动退回光标模式。

**全域体感功能手势**

- 三指伸展手势：静止保持 1 秒唤起半圆笔刷滚盘，循环切换普通笔、荧光笔、小草笔、小花笔等多款特效画笔；

- 拇指食指捏合手势：触发圆形橡皮擦，捏合移动可持续擦除涂鸦、素材，松开即刻退出擦除模式；

- 单掌五指全张手势：拖拽上层无限涂鸦画布整体平移，底层摄像头画面固定不变，握拳收拢即刻终止拖拽；

- 双手五指全张静置 2 秒：一键清空所有涂鸦、贴纸与特效素材，保留底层画面，过程中手势变动自动重置计时，防止误触发；

- 单手 L 型手势（食指伸直 + 拇指横撑）：生成仅作视觉装饰的阴影遮蔽框，双手隔空拖拽顶点可自由修改遮蔽为四边形、三角形、沙漏型等任意轮廓，静置 3 秒自动截取整张完整画布保存图片。

- 单掌张合序列动作（五指张开 → 缓慢捏拳）：唤起贴纸素材库，支持隔空选中、拖拽摆放、缩放调整素材样式。

#### 2.1.2 模板临摹功能

- 内置海量分层临摹模板库，涵盖卡通人物、小动物、星空风景、简约简笔画、节日主题等多类型素材；

- 模板以半透明底图固定于画布底层，用户可隔空沿轮廓描摹创作，零基础快速出片；

- 手势快捷切换模板：握拳切换上一张模板，摊手切换下一张模板，OK 手势调节模板透明度；

- 模板分难度等级，适配幼儿简笔画、二次元插画、风景素描不同创作需求；

- 完成临摹创作后自动弹出专属「临摹完成勋章」，解锁对应成就；

- 支持复刻临摹功能，可将社区公开作品一键设为临摹模板，二次创作学习。

#### 2.1.3 体感小游戏功能

复用手部关键点识别核心逻辑，内置多款轻量化体感小游戏，支持一键切换画布模式，丰富产品娱乐场景：

- **手势消消画**：画布随机散落彩色圆点，通过食指轨迹连线相同颜色圆点完成消除，计分闯关、难度递进；

- **隔空抓星星**：画面随机生成星星粒子，通过手掌摊开、握拳抓取星星，限时计分、榜单排行；

- **手势猜谜画画**：系统随机生成场景词语，用户限时手势绘画，结合图像匹配机制完成智能打分。

#### 2.1.4 动态贴纸与隔空贴图

- 比耶手势快速呼出贴纸面板，内置表情包、云朵、花朵、节日装饰等海量素材；

- 光标模式下单指拖拽贴纸摆放位置，握拳手势缩放贴纸大小，手掌滑动擦除多余贴纸；

- 依托贴纸功能，无绘画基础用户可快速拼接精美作品，彻底降低创作门槛。

#### 2.1.5 社交互动功能

- 作品社区生态：支持用户作品公开投稿、点赞、收藏、评论互动；

- 复刻临摹功能：一键引用社区优质作品作为临摹模板，实现创意传承与二次创作；

- 多维作品榜单：设置今日热门、本周最佳、创意榜单，展示优质创作内容；

- 品牌化分享能力：一键生成带 Air Doodle 水印、趣味边框、专属文案的分享海报；

- 自动录屏导出：通过 MediaRecorder 录制 3-10 秒绘画过程短视频，支持本地下载、多平台分享；

- 作品隐私管理：支持画作私密保存或公开投稿，兼顾隐私与社交需求。

#### 2.1.6 用户成就系统

- 打卡成就：连续打卡 3 天、7 天、30 天解锁对应勋章；

- 创作成就：根据完成画作数量、笔刷解锁数量、小游戏通关进度解锁层级成就；

- 成就激励：解锁成就可获取专属限定画笔、稀有贴纸模板，形成长期游玩激励机制。

#### 2.1.7 全栈业务功能

- 画布作品一键导出为图片格式，调用后端接口完成云端上传与存储；

- 数据库持久化存储作品信息、素材路径、用户成就、互动数据；

- 画廊页面通过 SSR 服务端渲染，实现作品分页展示、详情预览；

- 支持作品自主删除、内容管理、隐私状态修改；

- 完善用户信息管理、成就数据统计、互动数据记录功能；

- 页面全响应式适配，兼容电脑、手机各类终端屏幕。

#### 2.1.8 其他特色功能

- **手势文字涂鸦**：隔空手写线条自动规整为标准文字，支持隔空签名、手写祝福、创意文字创作；

- **双人同屏交互**：支持同一设备双人多手识别，实现双人协同绘画、小游戏对战；

- **节日限定活动**：端午、中秋、圣诞、新年等节点自动上线限定模板、专属笔刷与活动榜单；

- **AI 画作美化**：内置前端轻量化滤镜，支持一键将画作转换为动漫、油画、素描、胶片风格；

- **智能新手引导**：首次进入项目自动播放分步手势教程，完成手势识别校验后才可进入创作页面，降低上手难度。

### 2.2 非功能需求

- **实时性**：手势识别响应、绘画渲染延迟严格控制在 100ms 以内，画面流畅无卡顿、无拖影；

- **稳定性**：内置手势防抖、姿态过滤、置信度校验、计时防误触多重机制，长时间运行无崩溃、无内存泄漏、手势识别精准稳定；

- **兼容性**：适配 Chrome、Edge、Safari 等主流现代浏览器，跨电脑、手机终端正常运行，功能无缺失；

- **安全性**：原始摄像头视频流仅在前端本地处理，不上传服务器，全方位保护用户隐私；

- **可拓展性**：采用模块化解耦开发，手势、画笔、业务功能相互独立，支持后续新增功能迭代与硬件拓展；

- **沉浸感**：落笔光效、渐变线条、场景音效、移动端震动反馈全覆盖，打造沉浸式体感交互体验；

- **传播性**：海报、短视频生成耗时小于 3 秒，导出内容自带品牌标识与创意文案，适配社交传播场景。

## 3 系统总体架构设计

### 3.1 整体架构

本系统采用**分层软硬件架构 + Next.js 前后端一体化架构**，整体划分为五层架构，完整实现“硬件采集 - 前端计算 - 交互渲染 - 后端存储 - 页面展示”的全流程业务闭环，架构清晰、层级解耦、便于迭代维护。

1. **硬件采集层**：设备摄像头通过 WebRTC MediaDevices API 采集实时视频流，将光学图像转化为数字像素帧，为系统手势识别、画面渲染提供原始数据支撑。

2. **前端预处理层**：基于 Next.js 客户端组件处理原始视频帧，完成画面水平镜像翻转、分辨率缩放、画质优化，适配 AI 模型识别尺寸，降低设备算力消耗。

3. **Web AI 识别层**：通过 MediaPipe Hands WASM 轻量化模型，在浏览器本地离线解析手部 21 个关键点三维坐标，依托自研逻辑完成单手、双手、双人多手、序列动作识别，具备低延迟、高隐私、高精准度的优势。

4. **交互渲染层**：基于 HTML5 Canvas 实现多层分层渲染，分别承载固定底层画面、可漫游涂鸦图层、模板贴纸图层、手势关键点图层与动态特效图层，支撑全域无键鼠体感交互与无限画布创作。

5. **全栈服务层**：通过 Next.js API Route 提供各类业务接口，实现文件上传、数据增删改查、社区互动、成就管理等功能；依托 Prisma ORM 操作数据库，结合服务端组件完成画廊、社区页面 SSR 渲染。

### 3.2 技术栈详情

#### 前端技术栈（客户端）

- 核心框架：Next.js 15 (App Router)

- UI 框架：React 18 + Tailwind CSS，采用柔和渐变、童趣潮流的视觉风格，适配全年龄段用户

- 视频采集：WebRTC MediaDevices API，实现摄像头视频流实时获取与预处理

- AI 手势识别：MediaPipe Hands WASM，支持多手检测、自定义手势逻辑、防抖滤波优化

- 画布渲染：HTML5 Canvas 2D，实现分层无限画布渲染、动态特效叠加

- 图片处理：html-to-image，完成画布截图、海报合成

- 视频录制：MediaRecorder API，实现绘画过程短视频录制与导出

- 音效系统：Web Audio API，实现场景音效加载、播放与联动反馈

#### 后端技术栈（服务端）

- 接口服务：Next.js API Routes，统一提供各类业务接口服务

- 数据库 ORM：Prisma，标准化管理数据模型与数据库操作

- 数据库：开发环境采用 SQLite，生产环境可无缝切换为 PostgreSQL

- 文件存储：服务器本地文件系统，存储用户作品、模板、贴纸等静态资源

## 4 数据库设计

### 4.1 数据库选型

项目开发环境采用 SQLite 轻量数据库，无需额外部署服务，配置简单、部署便捷，适配本地开发调试场景。生产环境可无缝迁移至 PostgreSQL 数据库，适配高并发、大数据量的线上运营场景。项目基于 Prisma ORM 统一管理数据模型、数据库迁移与增删改查操作，实现代码与数据库解耦，提升开发效率与可维护性。

### 4.2 数据模型设计

在基础作品模型的基础上进行模块化拓展，新增用户、成就、点赞三大核心数据模型，支撑社区互动与成长激励体系，数据关联清晰、兼容原有业务逻辑，拓展性极强。

## 5 核心模块详细设计与实现

### 5.1 摄像头采集模块

本模块为系统硬件交互的核心入口，仅在 Next.js 客户端组件中运行，通过 WebRTC 标准 API 向用户申请摄像头权限，实时采集设备视频流数据。针对人体体感操作习惯，对原始画面进行水平镜像翻转处理，同时智能压缩视频分辨率，在保证识别精度的前提下降低 AI 模型算力消耗。模块内置完善的异常捕获机制，针对设备无摄像头、用户拒绝权限、设备占用等异常场景，给出可视化友好提示。底层摄像头画面永久固定渲染，不会随画布拖拽、清空、擦除操作变动，保证体感交互的基准画面稳定统一。

### 5.2 前端 AI 手势识别模块

模块基于 MediaPipe Hands WASM 轻量化 AI 模型开发，支持浏览器离线加载、本地独立运行，无需服务端算力支撑。系统逐帧解析视频画面，精准识别手部 21 个关键点三维坐标，搭载**防抖滤波、姿态过滤、置信度校验、计时防误触、深度分层判定**多重优化算法，彻底解决传统手势识别漂移、卡顿、误触发等行业痛点。

模块支持单手独立判定、左右手区分识别、双人多手同步检测、序列动作识别、长按计时触发等多元识别能力。自研分层手势优先级规则，通过优先级层级互斥机制，从逻辑根源杜绝手势指令冲突，实现全场景无键鼠、无触屏的纯体感交互。所有 AI 识别运算均在前端本地完成，原始视频数据不上传服务器，兼顾低延迟、高隐私、低算力消耗三大优势。

### 5.3 Canvas 涂鸦渲染模块

项目独创**四层分层 Canvas 独立渲染架构**，各层级互不干扰、独立刷新，完美适配无限延伸画布与全手势交互逻辑，具体分层设计如下：

-**底层（固定层）**：包含动态背景素材与固定摄像头预览画面，全程固定渲染，不参与拖拽、清空、擦除等操作，作为交互基准画面；

- **中层（创作层）**：核心可交互图层，包含无限可漫游涂鸦画布、半透明临摹模板、动态贴纸素材，支持手势整体拖拽平移、缩放编辑；

- **上层（识别层）**：实时渲染手部关键点、手势识别状态标注，为用户提供可视化交互反馈；

- **特效层（动态层）**：承载落笔拖尾光效、线条渐变发光、清空烟花、截图蒙版等动态特效，提升交互沉浸感。

系统实时追踪中指指尖坐标轨迹，通过贝塞尔曲线拟合算法生成平滑手绘线条，根据实时手势状态自动切换光标操控、绘画、擦除、画布漫游、素材编辑等模式，所有渲染操作均在客户端实时完成，交互零延迟、画面无卡顿。

### 5.4 特效画笔系统

在传统纯色画笔的基础上，项目自研全套体感动态特效笔刷体系，可通过三指手势快速切换，适配不同创作风格与场景，丰富创作趣味性：

- **星光笔**：沿绘画轨迹生成动态闪烁星点粒子，光影效果灵动；

- **彩虹笔**：线条颜色随绘画轨迹动态渐变，呈现七彩过渡效果；

- **烟花笔**：落笔瞬间触发微型烟花粒子扩散，动态效果炫酷；

- **水彩晕染笔**：线条边缘柔和模糊，模拟真实水彩扩散质感；

- **荧光霓虹笔**：线条自带高亮发光光晕，适配深色背景创作；

- **像素方块笔**：绘画轨迹离散为像素方块，打造复古像素风格画作；

- **爱心特效笔**：通过双手比心手势触发，绘画轨迹自动点缀动态爱心粒子。

### 5.5 体感小游戏模块

模块复用核心手部关键点识别逻辑，无需额外接入第三方算法，轻量化内置三款趣味体感小游戏，支持一键切换画布模式，打破单一绘画场景，有效提升用户留存时长：

1. **手势消消画**：系统在画布随机生成彩色圆点网格，通过识别食指指尖轨迹连线，匹配相同颜色相邻圆点即可消除计分，设置多关卡难度递进，兼顾趣味性与操作性；

2. **隔空抓星星**：通过帧循环随机生成星星粒子，识别手掌摊开、握拳的动作逻辑，判定手掌区域内的星星完成抓取计分，限时闯关、榜单排行，互动性极强；

3. **手势猜谜画画**：内置海量常用词语词库，随机抽取题目并启动倒计时，用户通过手势绘画作答，计时结束后通过图像匹配机制完成智能打分，兼具娱乐与创作锻炼价值。

### 5.6 社交分享模块

模块构建完整的作品社交与传播链路，实现创作、美化、导出、分享、互动全流程闭环：

- **实时录屏导出**：基于 MediaRecorder API 实时录制 Canvas 创作全过程，自动生成 3-10 秒短视频，支持 MP4 格式本地保存；

- **品牌海报合成**：将用户画作与专属边框、创意文案、Air Doodle 品牌水印自动合成精美分享海报，无需二次编辑；

- **多渠道分享**：支持图片、视频本地下载、链接复制、系统原生分享，适配全平台社交传播；

- **社区互动生态**：公开作品支持全网用户点赞、收藏、评论、复刻临摹，多维榜单展示优质作品，形成社区活跃氛围。

## 6 核心功能逻辑说明

本章定义项目**标准化、无冲突、全覆盖、可落地**的手势交互核心规范，适配无限延伸画布场景，通过明确的手势优先级、状态互斥、防误触机制，实现全程无键鼠、无触屏的纯体感操作，交互逻辑严谨、运行稳定。

### 6.1 基础状态优先级规则

系统设定固定手势执行优先级，高优先级手势自动覆盖低优先级手势，同一时间仅生效一种交互状态，从底层逻辑彻底杜绝手势冲突，优先级从高至低排序如下：

**双手组合指令 > 单手序列动作手势 > 功能指令手势 > 绘画模式 > 默认光标模式**

所有功能性手势均设置**时长阈值判定机制**，瞬时划过、短暂停留的手势不会触发功能；同时设置 0.7 识别置信度阈值，低于阈值的识别帧直接丢弃，全方位防止误操作。

### 6.2 两大基础核心模式

#### 6.2.1 光标模式（默认闲置模式）

**手势形态**：单独食指伸直，拇指、中指、无名指、小指完全弯曲收拢，手部姿态稳定无晃动。

**交互逻辑**：

- 指尖滑动：光标全域悬浮移动，等效鼠标滑动，可精准定位页面任意位置；

- 食指快速轻点：模拟鼠标左键单击，完成按钮触发、素材选中、作品确认等操作；

- 食指静止悬停 1 秒：唤起右键菜单，拓展更多操作功能；

- 食指+中指比耶滑动：实现页面、素材列表上下滚动；

- 本模式仅负责交互操控，不会生成任何绘画笔迹，为系统默认待机状态。

#### 6.2.2 绘画模式

**手势形态**：食指、中指并拢伸直，拇指、无名指、小指弯曲收拢，手部姿态稳定。

**交互逻辑**：

- 以中指指尖为精准笔尖基准，双指并拢移动即可持续生成平滑手绘线条；

- 任意时刻中指弯曲收回，即刻终止绘画逻辑，系统自动切换回光标模式；

- 绘画操作仅作用于上层涂鸦图层，底层摄像头基准画面永久不受影响。

### 6.3 单手功能手势规范

#### 6.3.1 笔刷切换手势（三指伸展）

**手势形态**：食指、中指、无名指三指伸直展开，拇指与小指弯曲收拢。

**触发规则**：手势静止保持 1 秒判定生效，短暂伸展、晃动不触发，避免误触。

**功能逻辑**：页面侧边弹出半圆笔刷滚盘，自动循环切换各类特效画笔，默认顺序为普通实心笔、荧光笔、小草笔、小花笔，新增自定义笔刷自动纳入循环队列；光标可隔空悬浮预览笔刷效果，点击确认切换。

#### 6.3.2 橡皮擦手势（双指捏合）

**手势形态**：拇指与食指指尖贴合或近距离夹持，剩余三指弯曲收拢。

**功能逻辑**：双指捏合瞬间激活圆形橡皮擦区域，保持捏合状态移动可持续擦除涂鸦线条、贴纸、特效素材；双指松开即刻退出擦除模式，自动切换为光标模式。

#### 6.3.3 无限画布拖拽手势（五指全张）

**手势形态**：单掌五指完全舒展张开，手部姿态平稳。

**功能逻辑**：手掌移动时，底层摄像头画面固定不变，仅上层涂鸦、模板、贴纸整体平移，实现无限画布全域漫游；五指握拳收拢即刻终止拖拽，画布定格当前位置。

#### 6.3.4 自定义取景截图手势（L 型手势）

**手势形态**：单手握L型，食指竖直伸直、拇指横向撑开形成直角 L 轮廓，中指、无名指、小指弯曲收拢；另一只手使用拇指 + 食指捏合手势，拖拽调整遮罩顶点。

**完整执行流程**：

1. 单手握持 L 型手势瞬间，画面生成可自由变形的阴影装饰遮蔽框；
   
2. 另一只手以拇指、食指捏合任意顶点拖拽，可自由调整顶点位置，遮蔽轮廓可变换为四边形、三角形、沙漏形等任意不规则多边形，该阴影遮蔽仅作为画面装饰、取景视觉标记，不会限制截图范围；
   
3. 双手保持 L 型 + 捏合调整手势稳定静置 3 秒，系统直接截取**整张完整画布画面**（包含底层摄像头画面、全部涂鸦、贴纸、遮罩装饰阴影）生成图片保存；
   
4. 任意一只手手势变形、抬手离开识别区域，立即销毁遮罩遮蔽，取消本次截图操作。

#### 6.3.5 贴纸素材库唤起手势（序列动作）

**触发动作**：单掌五指完全张开 → 缓慢匀速向内收拢握拳，整套连贯序列动作识别成功后触发功能。

**功能逻辑**：动作识别完成后弹出全景素材面板，包含本地贴纸、节日素材、社区优质素材；光标模式下可隔空点击粘贴素材，双指捏合可调整素材大小、位置、旋转角度。

### 6.4 双手最高优先级全局手势

#### 6.4.1 一键清空画布手势

**判定条件**：双手同时五指完全舒展张开，画面无大幅位移，手势稳定静置 2 秒。

**防误触机制**：计时过程中任意一只手晃动、握拳、位移，即刻重置计时，不触发清空操作。

**执行效果**：一键清空上层所有涂鸦、贴纸、特效、模板素材，底层摄像头基准画面完整保留，同时触发全屏烟花动画与专属音效反馈，操作感知清晰。

### 6.5 特色辅助手势功能

- **OK 手势体系**：左右手 OK 手势分别切换常规彩色画笔、特效画笔，双手同步 OK 手势切换素描画笔，多维度适配创作需求；

- **双手比心手势**：专属触发爱心特效画笔，绘画轨迹自动生成动态爱心粒子，打造创意浪漫画作；

- **握拳辅助手势**：单次快速握拳可切换上一张临摹模板，长按握拳可缩放选中素材大小，操作便捷高效。

### 6.6 双人同屏交互逻辑

依托 MediaPipe 多手同步检测能力，系统通过画面中线划分左右交互区域，独立识别两侧手掌手势，实现双人手势判定互不干扰。支持双人同时隔空绘画、各自操控素材、双人小游戏对战，完美适配亲子互动、情侣娱乐、线下团建等多人场景，大幅提升产品线下互动价值。

### 6.7 全栈业务流程

本项目完整业务流程覆盖用户从进入项目、创作互动、成果留存、社交传播、成长激励的全生命周期，结合前端体感交互与后端数据持久化能力，形成闭环业务逻辑，具体流程如下：

1. **初始化与权限校验流程**

用户进入项目首页后，系统自动触发摄像头权限申请，完成设备环境、浏览器兼容性校验。权限通过后启动摄像头视频流采集与AI模型预加载，同步弹出新手手势引导教程，用户完成基础手势识别校验后，方可进入涂鸦创作主界面，从源头规避操作障碍与设备适配问题。

2. **创作交互流程**

用户默认进入光标待机模式，可通过对应手势自由切换创作状态：既可开启自由涂鸦模式，搭配多款特效笔刷进行原创绘画；也可打开临摹模板库，选择对应难度的素材进行跟画创作，支持实时调节模板透明度适配创作节奏。创作过程中，可通过手势呼出贴纸素材、AI滤镜美化、音效特效等功能，丰富画作效果，同时支持无限画布拖拽漫游，突破固定创作边界。

3. **作品处理与存储流程**

用户完成创作后，可通过手势截图保存单帧画作，也可启动录屏记录完整创作过程。确认作品后，可自定义作品名称、设置公开 / 私密状态，前端将作品资源以 FormData 格式上传至后端服务。后端通过 UUID 生成唯一资源标识，完成图片静态资源存储，同时通过 Prisma ORM 将作品信息、创作参数、关联模板、滤镜类型、创作者信息等数据录入数据库，完成数据持久化。

4. **社区社交与互动流程**

公开作品将自动同步至画廊与社区页面，参与平台榜单排序。其他用户可浏览公开作品，完成点赞、收藏、评论互动，也可一键复刻优质作品为临摹模板，进行二次创意创作。系统实时统计作品互动数据，更新点赞量、浏览量等统计字段，支撑热门榜单、优质作品推荐等功能，形成社区创意流转生态。

5. **用户成长激励流程**

系统实时监听用户全量行为数据，包括每日打卡、创作数量、临摹通关、小游戏闯关、作品互动等行为。当用户行为满足对应成就解锁条件时，后端自动更新用户成就数据，前端即时弹出成就解锁弹窗，解锁专属笔刷、贴纸、勋章等限定权益，激励用户持续创作与活跃，提升用户复访率与产品粘性。

6. **内容分享与传播流程**

用户可随时对已完成作品进行二次处理，一键生成带品牌标识的精美海报与短视频素材，支持本地下载、多平台社交分享。分享内容自带标准化品牌文案与水印，无需二次编辑，实现低成本、高质感的自发传播，拓宽项目曝光渠道。

7. **个人内容管理流程**

用户可随时进入个人创作中心，查看全部历史作品、已解锁成就、创作数据统计，支持对个人作品进行隐私状态修改、删除管理、二次编辑复刻，自主管理个人创作内容资产，保障用户内容自主权与隐私安全。
